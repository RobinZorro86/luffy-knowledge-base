## 使用说明

### 什么时候写

- 每次执行手动任务（推文抓取、脚本运行、文件整理）
- 每次 cron 任务执行后（无论成功失败）
- 发现系统异常时
- 接受新任务并完成时

### 怎么写

每次记录包含：
```
## YYYY-MM-DD HH:MM — [任务简述]

**任务**：...
**结果**：✅ 成功 / ⚠️ 部分成功 / ❌ 失败
**详情**：...
**下次行动**：...
```

### 格式原则

- **时间倒序**：最新的在顶部
- **一句话说清楚**：任务是什么、结果是什么
- **失败必须写原因**：不能只写"失败"，要写为什么
- **关联性**：涉及推文入库、Wiki整理的，附上相关文件路径

---

## 系统状态基线

### Camofox
- **端口**：9377
- **检查命令**：`curl -s http://localhost:9377/tabs`
- **正常响应**：`{"running":true,"tabs":[...]}`
- **异常响应**：连接拒绝 / 超时 / `{"running":false}`

### Docker
- **检查命令**：`docker ps 2>/dev/null || echo "docker_not_available"`
- **正常响应**：列出运行中的容器
- **异常响应**：`docker: command not found` / 容器列表为空

### X Cookies
- **路径**：`/home/um870/x_cookies.txt`
- **格式**：Playwright JSON
- **baoyu路径**：`~/.local/share/baoyu-skills/x-to-markdown/cookies.json`
- **格式**：baoyu version:1 cookieMap

### MinerU API
- **Token**：`~/.hermes/config.yaml` 中的 `mineru_api_token`
- **服务**：`https://api.mineru.net`

### Hermes
- **Config**：`~/.hermes/config.yaml`
- **Profiles**：`~/.hermes/profiles/`
- **Cron**：`hermes cron list` 查看活跃任务
