# 推文入库标准规范

> 适用：所有从 X/Twitter 抓取并入库的推文文档
> 版本：v1.0 | 2026-04-14

---

## Frontmatter 必填字段（5项）

```yaml
---
title: "推文标题（原文或概括）"
author: "作者名 (@username)"
date: YYYY-MM-DD
source: https://x.com/username/status/XXXXXXXX
category: [分类目录]
tags: [tag1, tag2, tag3, ...]
status: curated
---
```

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 推文标题，双引号包裹 | `"为什么你的AI优先战略可能大错特错"` |
| `author` | 作者名 + @账号 | `"宝玉 (@dotey)"` |
| `date` | 发推日期，YYYY-MM-DD | `2026-04-13` |
| `source` | 完整推文URL | `https://x.com/dotey/status/2043953753921847582` |
| `category` | wiki分类目录 | `AI-Agent-Practices` |
| `tags` | 标签，≥3个 | `[AI-First, 软件工程, Harness-Engineering]` |
| `status` | `curated`（已整理）或 `raw`（原始） | `curated` |

### category 可选值

| category | 适用内容 |
|----------|---------|
| `AI-Agent-Practices` | Agent实践、方法论、工作流 |
| `AI-Agent-Architecture` | 多Agent架构、系统设计 |
| `AI-Education` | 儿童AI教育 |
| `Polymarket` | 预测市场、加密货币 |
| `Tool-Review` | 工具评测 |
| `Industry-Insights` | 行业洞察 |
| `Knowledge-Management` | 知识管理 |
| `Miscellaneous` | 其他 |

## 正文结构标准

### 必须包含的部分

```
1. 引言（>核心观点<）
   - 用引用块格式（> ... <）
   - 1-2句话概括推文最核心的观点

2. 核心内容
   - 按推文原有结构整理
   - 长推文（thread）按序号分段
   - 保留关键引用和金句

3. 摘要结尾（可选）
   - 一句话总结对Robin/我们的借鉴意义
   - 标注"## 延伸参考"关联已有wiki
```

### 引言块格式

```markdown
> 核心观点：（1-2句话概括）
```

### 标签规范（≥3个）

- 第1个：主题标签（如 `AI-First`, `Harness-Engineering`）
- 第2个：具体工具/技术（ 如 `Claude-Code`, `Obsidian`）
- 第3个：应用场景（如 `工作流重构`, `知识管理`）
- 其余：按需添加

---

## 质量门槛

入库前自检清单：

- [ ] frontmatter 5字段齐全（title/author/date/source/category）
- [ ] tags ≥ 3个
- [ ] 正文有 `>核心观点<` 引言块
- [ ] 推文正文 ≥ 200字（长推文）
- [ ] 末尾有 `## 延伸参考` 关联已有wiki
- [ ] 推文URL可访问（已验证）

## 文件命名规范

```
{年份}-{推文关键词}.md
```

示例：
- `2026-AI-First战略可能大错特错.md`
- `2026-约束先行-用好Agent最重要技巧.md`
- `2026-Obsidian-Claudian-AI终极个人知识库.md`

## 目录结构

```
~/wiki/twitter/
├── {作者名}/
│   ├── {年份}-{推文关键词}.md
│   └── ...
└── README.md  ← 本规范（可选）
```

作者名使用 Twitter 显示名，非@账号。

## 更新已有文档

如果已有文档frontmatter缺失，按以下优先级补全：

| 优先级 | 文件 | 操作 |
|--------|------|------|
| P0 | twitter/ 下所有文档 | 必须补全，2026-04-14前完成 |
| P1 | raw/tweets/ raw/articles/ | 已有完整frontmatter，跳过 |
| P2 | concepts/ 无前缀文档 | 增量补全 |
| P3 | zorro/团队知识库 | 长期工程，另行处理 |
