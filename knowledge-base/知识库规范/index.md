# 知识库规范

知识库建设与维护的标准操作规程。

## 文件清单

| 文件 | 说明 |
|------|------|
| Knowledge-Base-Compilation-SOP.md | 知识库编译标准操作流程 |
| 任务完成标准.md | 任务完成的评判标准 |

## 简介

知识库规范目录定义了知识库建设的核心 SOP：
- **Knowledge-Base-Compilation-SOP** - 知识编译流程规范
- **任务完成标准** - 判断任务是否完成的标准定义
- **推文入库标准** - 推特/推文内容收录规范

这些规范确保知识库内容的一致性和可维护性。
